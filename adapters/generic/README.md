# Generic Agent Adapter

Use this package with any LLM coding agent by giving it:

1. `frontend-design-ultra/SKILL.md`
2. `docs/AGENT-USAGE.md`
3. `docs/PARAMETERS.md`
4. relevant files from `frontend-design-ultra/references/`

Rule:

```text
Infer parameters, select references lazily, then produce the mode-specific output contract.
```

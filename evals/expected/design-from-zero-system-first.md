# Expected Output: Design From Zero System First

The answer must include:

- Research Read
- Evidence / Confidence
- Generated Design System
- Typography Package
- Color Direction
- Strategy Lane
- Signature Move
- UI Spec
- Asset Plan
- Verification Plan
- Forbidden Defaults

The answer must not:

- Start with component sections before brand/system.
- Use Inter, Roboto, Arial, or system UI without justification.
- Use generic blue/purple gradients without brand logic.
- Skip accessibility/performance gates.
